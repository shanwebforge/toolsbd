GitHub Keyboard Shortcuts

Essential GitHub keyboard shortcuts for developers - Repository Navigation, Code Review, Issue Management, Pull Request Controls, File Browsing, Comment Shortcuts, and productivity hotkeys for efficient collaboration and version control workflow

Shortcut Description
g then nGo to Notifications
g then iGo to Issues
g then pGo to Pull Requests
g then bGo to Your Repositories
g then dGo to Your Dashboard
g then cGo to Your Code
yPermalink to a file (copies URL with commit hash)
tActivate file finder
/Focus the search bar
.Open the repository in GitHub Codespaces (if available)
lOpen language filter in file list
cCreate a new issue or pull request
eEdit issue or comment
rReload the page
mToggle reaction emoji picker
TabIndent comment
Shift + TabOutdent comment
uGo to the conversation tab of a pull request
iGo to the commits tab of a pull request
dGo to the files changed tab of a pull request
nNext comment or item
pPrevious comment or item
Shift + ?Open shortcut help

GitHub / Git All Commands

Command Description (Bangla)
git initনতুন একটি গিট রেপোজিটরি শুরু করুন
git clone [url]রিমোট রেপো কপি করুন লোকাল মেশিনে
git statusফাইলের বর্তমান অবস্থা দেখায়
git add [file]ফাইলকে স্টেজিং এরিয়াতে রাখুন
git add .সব পরিবর্তন স্টেজ করুন
git commit -m "message"পরিবর্তন রেকর্ড করুন একটি বার্তাসহ
git logপূর্ববর্তী কমিট গুলো দেখায়
git branchব্রাঞ্চ লিস্ট করে এবং বর্তমানে কোনটিতে আছেন তা দেখায়
git branch [name]নতুন ব্রাঞ্চ তৈরি করুন
git checkout [name]অন্য ব্রাঞ্চে যান
git merge [branch]অন্য ব্রাঞ্চ থেকে কোড মার্জ করুন
git pullরিমোট রেপো থেকে কোড আনুন ও মার্জ করুন
git pushলোকাল কমিট গুলো রিমোটে পাঠান
git remote -vরিমোট রেপো লিস্ট দেখায়
git remote add origin [url]রিমোট রেপো সেট করুন
git config --global user.name "নাম"গ্লোবাল ইউজারনেম সেট করুন
git config --global user.email "ইমেইল"গ্লোবাল ইমেইল সেট করুন
git reset --hardসব পরিবর্তন বাতিল করে পূর্ব অবস্থায় ফিরিয়ে নেয়
git stashবর্তমান কাজ সাময়িকভাবে সংরক্ষণ করুন
git stash popসেই সংরক্ষিত কাজ আবার ফিরিয়ে আনুন
git fetchরিমোট থেকে নতুন আপডেট আনে, কিন্তু মার্জ করে না
git diffপরিবর্তনসমূহ তুলনা করে দেখায়
git rm [file]ফাইল রেপো থেকে মুছে ফেলুন
git mv [old] [new]ফাইলের নাম পরিবর্তন করুন
git tagকমিটগুলোতে ট্যাগ অ্যাড করুন (রিলিজে কাজে লাগে)
git showএকটি কমিট বা ট্যাগের বিস্তারিত দেখুন
